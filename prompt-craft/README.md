# Prompt Craft (提示词修炼)

从零开始构建高质量、生产级的提示词。通过极致压缩（语义点亮）与结构化定义，消除语义空洞，实现精准的意图对齐与泛化能力。

## 技能简介
本技能旨在帮助开发者从底层逻辑出发，掌握“得其意而忘其言”的提示词编写技巧。通过将模糊的“描述”转化为精准的“定义”，点亮模型向量空间中的特定坐标，从而在提升 Agent 稳定性的同时，赋予其极强的泛化能力。

## 文件结构
- **SKILL.md**: 核心指令集与方法论。详细介绍了 7 大修炼维度（哲学、压缩、结构、注意力、角色、金阶技巧、格式适配）以及标准化的提示词构建工作流。
- **REFERENCE.md**: 快速查阅手册。包含脱水修炼对照表、描述转定义案例、主流格式模板（Markdown/Lisp/XML）以及质量自检清单。
- **EXAMPLES.md**: 实战案例库。展示了客服智能体（结构化）、创意卡片（Lisp 压缩）、复杂分析（混合模式）及科普降维（Few-shots）等典型应用场景。

## 使用指南
1. **理解本质**: 深入挖掘任务背后的底层逻辑与 Know-how。
2. **选择适配**: 根据目标模型（GPT/Claude/Gemini）与任务复杂度选择最佳格式。
3. **分配压缩包**: 选定核心角色，必要时进行维度解压（注入背景、规则等）。
4. **构建骨架**: 利用 Markdown 或 XML 标签建立清晰的语义聚拢结构，防止注意力漂移。
5. **压缩脱水**: 删除冗余描述，填入高密度的定义词，确保注意力权重集中在核心意图。
6. **验证优化**: 参考 `REFERENCE.md` 进行自检，并在实际交互中迭代。

## 与 prompt-refactor 的关系
两者为互补关系，共同构成提示词工程的全生命周期：
- **prompt-craft (修炼)**: 侧重于**从零开始**构建复杂 Agent 或核心方法论，建立稳定的底层架构。
- **prompt-refactor (重构)**: 侧重于对**现有提示词**进行优化、修补或性能提升，解决输出不稳定或注意力漂移等具体问题。
