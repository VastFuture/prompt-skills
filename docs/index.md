## 我应该用哪个？

| 场景 | 推荐技能 | 做什么 |
|------|----------|--------|
| 不知道从哪开始写 | `prompt-template` | 六要素模板，套进去就有完整提示词 |
| 已经有初稿想优化 | `prompt-optimize` | 匹配57种框架，生成可执行prompt |
| 提示词太长太乱 | `prompt-refactor` | 重构生产级提示词，解决碎片化问题 |
| 想提升现有skill | `skill-optimize` | 优化skill结构、触发、工作流 |

## 技能关系图

```
从零构建 ──→ prompt-craft（修炼）
              ↓
框架优化 ──→ prompt-optimize（框架匹配与生成）
              ↓
持续迭代 ──→ prompt-refactor（重构）
              ↓
技能质量 ──→ skill-optimize（skill 结构优化）
              ↓
          回到 craft 重新审视底层逻辑
```
