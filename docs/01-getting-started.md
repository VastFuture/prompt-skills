# 5分钟快速入门

欢迎来到 Prompt Skills！本指南将帮助你快速上手提示词工程。

---

## 什么是提示词技能？

提示词技能（Prompt Skills）是一套 AI Agent 的提示词工程技能集。它帮助你：

- 从零构建高质量提示词
- 基于任务场景匹配最佳框架
- 优化和重构现有提示词
- 提升 AI 输出质量和稳定性

---

## 三步走完提示词生命周期

```
① 写 ──→ ② 优化 ──→ ③ 重构
```

### ① 写（从零构建）

使用 `prompt-craft` 从零构建高质量、生产级提示词。

**适用场景**：需要创建一个全新的提示词
**核心能力**：极致压缩与结构化定义，消除语义空洞

### ② 优化（框架匹配）

使用 `prompt-optimize` 匹配合适框架，生成更清晰的 prompt。

**适用场景**：已有初稿想优化，或需要按框架重写
**核心能力**：57种提示词框架，复杂度与领域双维度匹配

### ③ 重构（系统性改进）

使用 `prompt-refactor` 对生产级提示词进行系统性重构。

**适用场景**：提示词太长太乱、输出不稳定
**核心能力**：诊断碎片化、模板腔等问题，5条改写铁律

## 你属于哪种情况？

### 不知道从哪开始写

**推荐**: `prompt-template`

> "我有个模糊的想法，但不知道怎么组织成完整的提示词"

六要素模板（Role / Background / Task / Requirements / Example / Reflection），套进去就有结构完整的提示词。

### 已经写了一个但效果不好

**推荐**: `prompt-score`

> "我写了一个提示词，但 AI 输出不太对劲"

结构化评分专家，给你的提示词打分（1-10），并给出具体改进建议。（1-10），并给出具体改进建议。

### 提示词太长了太乱了

**推荐**: `prompt-refactor`

> "我的提示词有500多行，自己都看不懂了"

系统性重构，解决规则碎片化、输出不稳定、AI 模板腔等问题。

### 在 Coze 上写机器人

**推荐**: `prompt-coze`

> "我在 Coze.cn 搭智能体，卡在变量定义和输出格式上"

专为 Coze 平台设计，含输入/输出变量定义与 JSON 对齐。

## 第一次使用示例

假设你想写一个产品介绍助手：

**Step 1**: 使用 `prompt-template` 快速构建模板

```
你是一个 [产品营销专家]，
我正在 [为我们的SaaS产品写推广文案] 下，
需要你执行 [撰写一篇产品介绍文案]。
请你按照 [专业但亲切的语气，800字以内，包含核心卖点和用户收益]（包括风格、格式、长度等）来完成，
这里有一个 [示例] 供你参考。
最后，请 [反思] 你的输出是否满足要求。
```

**Step 2**: 使用 `prompt-optimize` 匹配框架深化（可选）

如果你需要更复杂的营销框架，可以让它匹配 `BAB` 或 `SPEAR` 框架。

**Step 3**: 使用 `prompt-refactor` 打磨优化（如果效果不好）

检查是否有多余的解释层、重复规则、边界泄漏等问题。
